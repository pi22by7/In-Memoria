# Change Log

All notable changes to the "In Memoria Visualizer" extension will be documented in this file.

## [0.1.0] - 2025-01-07

### Added
- Initial release of In Memoria Visualizer
- Intelligence Dashboard with metrics and overview
- Hierarchical Tree Views for project intelligence, patterns, work session, and insights
- Interactive Relationship Graph with force-directed layout
- Pattern Explorer with confidence-based grouping
- File Intelligence panel for analyzing specific files
- Feature Router for AI-powered file routing
- MCP client integration with In Memoria server
- Auto-connect and auto-refresh capabilities
- Search concepts command
- Route feature to files command
- Comprehensive configuration options

### Features
- **Project Intelligence Tree**: Tech stack, entry points, key directories, architecture
- **Patterns Tree**: Learned patterns with confidence scores and examples
- **Work Session Tree**: Current files, tasks, and decisions tracking
- **AI Insights Tree**: Contributed insights from AI agents
- **Dashboard**: Real-time metrics and status overview
- **Relationship Graph**: Visual concept relationships and dependencies
- **Pattern Explorer**: Deep dive into coding patterns
- **File Intelligence**: Per-file semantic analysis
- **Feature Router**: Natural language feature-to-file routing

### Commands
- Show Intelligence Dashboard
- Show Relationship Graph
- Show Pattern Explorer
- Show Work Session
- Show File Intelligence
- Show Feature Router
- Learn Codebase
- Search Concepts
- Route Feature to Files
- Refresh Intelligence
